## Chapitre 1 — Inscription & Premiers Pas

### 1.1 Créer votre compte

1. **Téléchargez Stockman** depuis le Play Store (Android) ou l'App Store (iOS).
2. Ouvrez l'application et appuyez sur **« S'inscrire »**.
3. **Choisissez votre profil** :
   - 🏪 **Commerçant** — Vous gérez un ou plusieurs commerces.
   - 🚚 **Fournisseur** — Vous vendez en gros à des commerçants.
4. Remplissez vos informations :
   - **Nom** : Votre nom ou le nom de l'entreprise.
   - **Email** : Utilisé pour la connexion (ex : `boutique.amina@gmail.com`).
   - **Mot de passe** : Minimum 6 caractères.
   - **Téléphone** : Votre numéro WhatsApp de préférence.
   - **Pays & Devise** : Sélectionnez votre pays (ex : Sénégal → FCFA, France → EUR).
5. **Choisissez votre plan** (Starter, Pro ou Enterprise) — L'essai est **gratuit pendant 90 jours** sans carte bancaire.
6. Appuyez sur **« Créer mon compte »**.
7. Un **code de vérification à 6 chiffres** est envoyé sur votre téléphone. Saisissez-le pour valider.

> 💡 **Conseil** : Choisissez un email que vous consultez régulièrement. C'est aussi l'adresse pour récupérer votre mot de passe.

### 1.2 Découvrir l'interface

Après l'inscription, vous arrivez sur le **Tableau de Bord**. En bas de l'écran, vous voyez la barre de navigation :

| Icône | Onglet | Rôle |
|-------|--------|------|
| 🏠 | **Dashboard** | Vue d'ensemble de votre activité |
| 📦 | **Produits** | Votre catalogue et votre stock |
| 🛒 | **POS** | Pour réaliser les ventes |
| 🔔 | **Alertes** | Les notifications importantes |
| ⚙️ | **Plus** | Comptabilité, CRM, Fournisseurs, Paramètres |

### 1.3 Sécuriser l'accès (PIN & Biométrie)

1. **Plus** → **Paramètres** → **Sécurité** → **Code PIN**.
2. Créez un code à 4 chiffres pour verrouiller l'application.
3. Après avoir activé le PIN, vous pouvez activer la **biométrie** (empreinte digitale ou Face ID) pour un accès encore plus rapide.
4. Si vous vous déconnectez, la reconnexion biométrique reste possible tant que cette option est activée.

> 💡 **Conseil** : Si vous partagez votre téléphone avec des employés, le PIN empêche l'accès non autorisé aux données sensibles.

## Chapitre 2 — Ajouter vos Produits ⭐

> **C'est l'étape la plus importante !** Sans produits, l'application ne peut rien faire. Prenez le temps de bien remplir cette partie.

### 2.1 Ajouter un produit manuellement

1. Appuyez sur l'onglet **📦 Produits** en bas.
2. Appuyez sur le bouton **« + »** (en bas à droite).
3. Remplissez les informations :

| Champ | Obligatoire | Exemple | Explication |
|-------|:-----------:|---------|-------------|
| **Nom** | ✅ | Riz Brisure 25kg | Le nom tel que vous le connaissez |
| **Prix d'achat** | ✅ | 8 500 FCFA | Ce que vous payez au fournisseur |
| **Prix de vente** | ✅ | 10 000 FCFA | Ce que le client paie |
| **Quantité** | ✅ | 50 | Combien vous en avez en stock |
| **Stock minimum** | Recommandé | 10 | Stockman vous alertera sous ce seuil |
| **Unité** | Recommandé | Sac / Pièce | L'unité de vente |
| **Catégorie** | Recommandé | Alimentation | Pour organiser vos produits |
| **SKU / Code** | Optionnel | RIZ-25K-001 | Un code unique |
| **Photo** | Optionnel | 📷 | Photo du produit |
| **Description** | Optionnel | Riz local... | Détails du produit |
| **N° de lot** | Optionnel | LOT2026-001 | Pour produits avec numéro de lot |
| **Date de péremption** | Optionnel | 2026-06-15 | Pour les produits périssables |

4. Appuyez sur **« Enregistrer »**.

> 💡 **Exemple** : Amina tient une boutique à Dakar. Elle commence par ajouter ses 20 produits les plus vendus : Riz, Huile, Sucre, Lait en poudre, etc.

### 2.2 Lots et Dates de Péremption

Pour les produits alimentaires ou pharmaceutiques, la gestion par lots est essentielle.

1. Lors de l'ajout ou modification d'un produit, renseignez :
   - **Numéro de lot** (ex : `LOT2026-001`)
   - **Date de péremption** (format YYYY-MM-DD, ex : `2026-06-15`)
2. Stockman enverra automatiquement une alerte quand le lot expire dans les **7 jours**.
3. Les lots prochain à expirer sont visibles depuis le **Dashboard** → section **« Alertes Péremption »**.

> ⚠️ **Important** : Si vous avez plusieurs lots du même produit avec des dates différentes, créez le produit une fois et notez le lot dans les détails. La sortie de stock suit la logique FIFO (premier entré = premier sorti) automatiquement.

### 2.3 Variantes de Produits

Pour un produit disponible en plusieurs versions (tailles, couleurs, etc.) :

1. Ouvrez la fiche d'un produit → **« Variantes »**.
2. Appuyez sur **« + Ajouter une variante »**.
3. Donnez un nom à la variante (ex : `XL`, `Rouge`, `500ml`).
4. Chaque variante a son propre stock indépendant.

> 💡 **Exemple** : Un t-shirt vendu en tailles S, M, L, XL. Vous créez un seul produit "T-Shirt Stockman" avec 4 variantes. Chaque taille a son propre stock.

### 2.4 Historique des Prix

Pour consulter l'évolution des prix d'un produit :

1. Ouvrez la fiche du produit → Onglet **« Historique »** → **« Prix »**.
2. Vous voyez toutes les modifications de prix avec date et ancienne valeur.

Ceci est utile pour négocier avec vos fournisseurs ou analyser vos marges dans le temps.

### 2.5 Étiquettes PDF

Générez des étiquettes à imprimer pour vos rayons :

1. Ouvrez la fiche d'un produit → **« Étiquette PDF »**.
2. L'étiquette contient : nom, prix de vente, QR code, et code-barres.
3. Partagez via WhatsApp ou imprimez directement.

### 2.6 Organiser vos produits par emplacement

Si la gestion des emplacements est disponible sur votre compte, vous pouvez structurer votre stock de façon beaucoup plus claire.

**Exemples d'organisation possibles :**
- `Allée 1 / Niveau 2 / Étagère 7`
- `Zone froide / Rayon B / Bac 3`
- `Réserve / Mur gauche / Colonne 4`

**Comment faire :**

1. Ouvrez la fiche d'un produit.
2. Repérez le champ **« Emplacement »**.
3. Sélectionnez l'emplacement déjà créé ou appuyez sur **« Gérer »** pour construire votre structure.
4. Enregistrez le produit.

**Bon à savoir :**
- vous pouvez créer un emplacement isolé pour un besoin rapide ;
- vous pouvez aussi générer une structure complète d'un seul coup au lieu de tout créer un par un ;
- le chemin complet de l'emplacement vous aide ensuite à ranger, compter et retrouver vos articles plus vite.

### 2.7 Mouvements de Stock (Entrées / Sorties)

Pour ajuster le stock sans créer de vente ou de commande :

1. Dans **📦 Produits**, appuyez sur le produit voulu.
2. Appuyez sur **« +Stock »** (entrée) ou **« -Stock »** (sortie).
3. Précisez la quantité et la **raison** :
   - Réception fournisseur
   - Ajustement inventaire
   - Perte / Casse
   - Vente directe

Ces mouvements sont enregistrés dans le **Grand Livre** (voir Chapitre 5).

### 2.8 Importer des produits en masse (CSV)

Si vous avez beaucoup de produits (50, 100 ou plus), l'import CSV est beaucoup plus rapide.

**Préparez votre fichier Excel/CSV :**

```csv
Nom,Prix achat,Prix vente,Quantité,Catégorie,Stock minimum,Emplacement
Riz Brisure 25kg,8500,10000,50,Alimentation,10,Allée 1 / Niveau 2 / Étagère 7
Huile Niinal 1L,1200,1500,80,Alimentation,20,Zone froide / Rayon B
Sucre 1kg,600,750,100,Alimentation,15,Réserve / Bac 3
```

**Importez dans Stockman :**

1. Allez dans **📦 Produits**.
2. Appuyez sur l'icône **« Importer »** (en haut à droite).
3. Sélectionnez votre fichier CSV.
4. **Mapping intelligent** : Stockman détecte automatiquement les colonnes. Vérifiez chaque correspondance.
5. **Aperçu** : Vérifiez vos produits avant de valider.
6. Appuyez sur **« Lancer l'importation »**.

### 2.9 Scanner un code-barres

Pour les produits avec un code-barres :

1. Dans **📦 Produits**, appuyez sur l'icône **scanner** 📸.
2. Visez le code-barres du produit.
3. Si le produit existe déjà, sa fiche s'affiche. Sinon, un formulaire vierge s'ouvre avec le code pré-rempli.

### 2.10 Inventaire Physique

Pour compter votre stock réel et corriger les écarts :

1. Dans **📦 Produits** → appuyez sur le bouton **« Mode Inventaire »** (icône liste en haut).
2. Pour chaque produit, entrez la quantité réelle comptée.
3. Stockman compare la quantité théorique et la quantité réelle.
4. Validez → les stocks sont mis à jour et un **rapport d'inventaire PDF** est généré.

> 💡 **Conseil professionnel** : Faites votre inventaire en **début ou fin de journée** quand il n'y a pas de ventes en cours. Assignez la tâche à un employé de confiance.

### 2.11 Scan Rapide par Lot

Pour compter ou enregistrer plusieurs produits d'un coup :

1. Appuyez sur **« Scan Rapide (Lot) »**.
2. Scannez tous vos produits un par un — chaque scan s'ajoute à la liste.
3. Quand vous avez terminé, choisissez l'action :
   - **Inventaire rapide** : Marquer ces articles comme présents.
   - **Réapprovisionnement auto** : Générer des bons de commande pour les manquants.

### 2.12 Supprimer plusieurs produits

Quand vous devez nettoyer un grand catalogue :

> Bon à savoir : le bouton **Tout sélectionner** prend en compte toute la liste visible, même si les produits se chargent sur plusieurs pages.

1. Dans **📦 Produits**, activez la sélection multiple.
2. Sélectionnez les produits concernés.
3. Appuyez sur l'action de suppression et confirmez.
4. Les produits quittent la liste immédiatement. Stockman les déplace ensuite vers la **corbeille** en arrière-plan et affiche l'avancement si le traitement est encore en cours.

> 💡 **Bon à savoir** : La suppression groupée n'efface pas définitivement les produits. Ils restent restaurables depuis la corbeille tant qu'ils n'ont pas été supprimés définitivement.

## Chapitre 3 — Le Tableau de Bord

Le tableau de bord est la première chose que vous voyez en ouvrant l'application. Il affiche un résumé en temps réel de votre activité.

### 3.1 Les indicateurs clés (KPI)

| Indicateur | Signification | Exemple |
|------------|---------------|---------|
| **CA du jour** | Montant total des ventes aujourd'hui | 125 000 FCFA |
| **Ventes jour** | Nombre de transactions | 18 ventes |
| **Ventes mois** | CA des 30 derniers jours | 2 850 000 FCFA |
| **Valeur stock** | Valeur de tout votre stock restant | 5 200 000 FCFA |

### 3.2 Les sections du tableau de bord

- **📊 Ventes Récentes** — Les 5 dernières ventes avec montant et heure.
- **⚠️ Produits Critiques** — Les produits en stock bas ou en rupture. Appuyez sur **« Commander »** pour lancer un réapprovisionnement.
- **📈 Analyse ABC** — Classement de vos produits par rentabilité :
  - **Classe A** (🟢) : Vos produits « stars » — 80% de votre chiffre d'affaires.
  - **Classe B** (🟡) : Produits moyens — 15% du CA.
  - **Classe C** (🔴) : Produits à faible rotation — 5% du CA.
- **🔄 Mouvements (30j)** — Entrées et sorties de stock sur le mois.
- **⏰ Alertes Péremption** — Produits dont le lot expire dans les 30 jours.
- **💡 Réapprovisionnement Intelligent** — L'IA analyse vos ventes et suggère les produits à commander avec les quantités optimales.
- **🤖 Résumé IA du Jour** — L'IA analyse votre journée et vous donne un résumé motivant.

> 💡 **Astuce** : Vous pouvez **personnaliser** les sections affichées en appuyant sur l'icône ⚙️ en haut à droite du dashboard.

### 3.3 Rapport journalier et partage

Selon votre formule, une section supplémentaire peut afficher :
- CA du jour comparé à hier (avec variation en %).
- Nombre de ventes comparé à hier.
- Top 3 produits du jour.
- Bouton **« Partager »** pour envoyer le résumé par WhatsApp.

### 3.4 Rappels Intelligents

Depuis le Dashboard, Stockman affiche des rappels automatiques basés sur votre activité :

| Rappel | Déclencheur |
|--------|-------------|
| **Réapprovisionnement** | Produits sous le stock minimum |
| **Produits dormants** | Sans vente depuis X jours |
| **Livraisons en retard** | Commande fournisseur non reçue |
| **Recouvrement de dettes** | Clients avec dettes en attente |
| **Anniversaires clients** | Date d'anniversaire dans les 7 jours |
| **Rapport mensuel** | Fin du mois → générez votre bilan |

Configurez ces rappels dans **Plus** → **Paramètres** → **Règles de rappels**.

## Chapitre 4 — Vendre avec le Point de Vente (POS)

Le POS est votre caisse enregistreuse numérique.

### 4.1 Réaliser une vente

1. Appuyez sur l'onglet **🛒 POS**.
2. **Cherchez le produit** : Tapez son nom dans la barre de recherche, ou scannez son code-barres.
3. **Ajoutez au panier** : Appuyez sur le produit → il s'ajoute au panier à droite.
4. **Modifiez la quantité** si besoin (boutons + et −).
5. Appuyez sur **« Valider la vente »**.

### 4.2 Choisir le type de transaction

Lors d'une vente, vous indiquez **comment le client a payé** afin d'avoir un suivi comptable précis. Stockman ne traite pas les paiements directement — il enregistre simplement le type pour vos rapports.

| Type | Icône | Quand l'utiliser |
|------|-------|-----------------|
| **Espèces** | 💵 | Le client paie en cash |
| **Mobile Money** | 📱 | Orange Money, Wave, Free Money… |
| **Carte** | 💳 | Paiement par carte bancaire |
| **Virement** | 🏦 | Virement bancaire |
| **Crédit** | 📝 | Le client n'a pas encore payé (dette) |

> 💡 **À quoi ça sert ?** Ces types servent à **ventiler votre chiffre d'affaires** par mode d'encaissement dans vos rapports comptables. Vous pouvez ainsi savoir quelle part de vos ventes est en cash vs Mobile Money vs crédit.

### 4.3 Associer un client à la vente

1. Avant de valider, appuyez sur **« Client »**.
2. Sélectionnez un client existant ou créez-en un nouveau en quelques secondes.
3. Si le client paie à crédit, la dette est automatiquement enregistrée dans son compte.
4. Les **points de fidélité** sont crédités automatiquement.

> 💡 **Exemple** : Moussa achète 3 sacs de ciment à crédit. Vous sélectionnez Moussa, choisissez « Crédit ». La dette de 12 600 FCFA apparaît automatiquement dans la fiche de Moussa.

### 4.4 Appliquer une remise

1. Appuyez sur **« Remise »** avant de valider.
2. Choisissez le type : **Pourcentage** (ex : 10%) ou **Montant fixe** (ex : 500 FCFA).
3. La remise s'applique au total du panier.

### 4.5 Produits souvent achetés ensemble

Lors d'une vente, Stockman affiche sous le panier des suggestions **« Souvent achetés ensemble »** basées sur l'historique de vos clients. Très utile pour augmenter le panier moyen.

### 4.6 Le reçu digital

Après chaque vente, un reçu est généré automatiquement. Vous pouvez :
- **L'envoyer par WhatsApp** au client.
- **L'envoyer par Email**.
- **L'imprimer** si vous avez une imprimante connectée.
- Partager un **lien QR code** que le client peut scanner pour consulter son reçu en ligne.

### 4.7 Terminaux de caisse

Si cette gestion est disponible sur votre compte, vous pouvez configurer plusieurs **terminaux de caisse** dans une même boutique :

1. **POS** → icône terminal en haut → **« Sélectionner le terminal »**.
2. Choisissez le terminal actif (ex : Caisse 1, Caisse 2).
3. Chaque caisse enregistre ses propres sessions et rapports.

## Chapitre 5 — Comptabilité & Finances

### 5.1 Accéder à la comptabilité

Appuyez sur **Plus** (barre du bas) → **Comptabilité**.

### 5.2 Les indicateurs financiers

| Indicateur | Formule | Exemple |
|------------|---------|---------| 
| **Chiffre d'Affaires (CA)** | Total des ventes | 2 850 000 FCFA |
| **Coût d'achat (COGS)** | Prix d'achat × quantités vendues | 1 900 000 FCFA |
| **Marge brute** | CA − Coût d'achat | 950 000 FCFA |
| **Total charges** | Somme des dépenses | 350 000 FCFA |
| **Bénéfice Net** | Marge brute − Total charges | 600 000 FCFA |

Pour analyser les pertes et les dépenses, touchez directement les cartes KPI **« Pertes »** et **« Dépenses »** en haut de l'écran : le détail s'ouvre dans la modale d'information.
Les ratios affichés dans ces détails (poids des dépenses, marge nette, taux d'activité) sont déjà exprimés en pourcentage du chiffre d'affaires.

### 5.3 Enregistrer une dépense

1. Appuyez sur **« Nouvelle dépense »**.
2. Choisissez la catégorie : Loyer, Salaire, Transport, Électricité, Eau, Internet, Marchandise, Autre.
3. Entrez le montant et une description.
4. Appuyez sur **« Confirmer »**.

> 💡 **Exemple** : Vous payez votre loyer de 150 000 FCFA. Catégorie = « Loyer », Montant = 150 000, Description = « Loyer boutique Sandaga Mars 2026 ».

### 5.4 Choisir la période d'analyse

Utilisez les filtres en haut de l'écran :
- **7 jours** | **30 jours** | **90 jours** | **1 an** | **Personnalisé** (choisir les dates).

### 5.5 Graphique d'évolution du CA

Le graphique montre l'évolution jour par jour de votre chiffre d'affaires. Utilisez-le pour repérer les bons et les mauvais jours.

### 5.6 Analyse par mode de paiement

Visualisez votre CA ventilé par mode de paiement :
- Espèces, Mobile Money, Carte, Virement, Crédit.

Utile pour savoir si vos clients paient surtout en cash ou en Mobile Money.

### 5.7 Pertes & Casses

Enregistrez vos pertes de marchandise :
1. **Comptabilité** → **« Pertes / Casses »** → **« + »**.
2. Sélectionnez le produit, entrez la quantité perdue et la raison.
3. La valeur est déduite de votre stock et comptabilisée comme perte.

### 5.8 Créer une Facture PDF professionnelle

Stockman vous permet de créer de vraies factures à envoyer à vos clients professionnels :

1. **Comptabilité** → **« Facture »** → **« Créer une facture »**.
2. Renseignez les informations du client (nom, adresse).
3. Ajoutez des **lignes** : description, quantité, prix unitaire.
4. Ajoutez la **TVA** si applicable.
5. Appuyez sur **« Générer PDF »**.
6. Partagez par **Email** ou **WhatsApp**.

> 💡 **Exemple** : Vous livrez 50 sacs de ciment à un chantier de construction. Vous créez une facture professionnelle avec votre logo, les détails, la TVA et vous l'envoyez par email au responsable des travaux.

### 5.9 Le Grand Livre (Journal des Mouvements)

Le Grand Livre est le registre complet de **tous les mouvements de stock et financiers** de votre boutique. C'est l'équivalent numérique du « cahier de caisse ».

**Pour accéder au Grand Livre :**

1. **Dashboard** → bouton **« Historique »** (en haut à droite) ou icône téléchargement.
2. Vous voyez la liste de **toutes les opérations** classées par date :

| Opération | Exemple |
|-----------|---------|
| **Vente POS** | -3 Riz + 15 000 FCFA encaissé |
| **Entrée stock** | +50 Huile (réception fournisseur) |
| **Sortie manuelle** | -2 Sucre (casse) |
| **Dépense** | -150 000 FCFA (loyer) |
| **Ajustement inventaire** | Correction stock Farine : 45 → 42 |

**Filtres disponibles :**
- Par module (stock, caisse, comptabilité)
- Par produit
- Par période

**Export CSV :**
Appuyez sur l'icône téléchargement pour exporter le Grand Livre en fichier Excel. Idéal pour votre comptable ou vos déclarations fiscales.

> 💡 **Conseil** : Consultez le Grand Livre **chaque semaine** pour repérer les anomalies (sorties non justifiées, erreurs de saisie, etc.).

### 5.10 Performance par produit

La section **« Performance Produit »** dans la comptabilité vous montre pour chaque produit :
- Quantité vendue sur la période
- CA généré
- Marge brute
- Rentabilité en %

### 5.11 Exporter vos données

- **Export CSV** : Appuyez sur **« Export CSV »** pour télécharger un fichier Excel avec vos ventes et dépenses.
- **Rapport PDF** : Générez un rapport PDF complet avec tous les KPIs.

## Chapitre 6 — Gérer vos Clients (CRM)

### 6.1 Accéder au CRM

Appuyez sur **Plus** → **CRM & Fidélité**.

### 6.2 Ajouter un client

1. Appuyez sur **« Nouveau Client »**.
2. Remplissez : **Nom**, **Téléphone** (optionnel), **Email** (optionnel), **Date d'anniversaire** (optionnel), **Notes** (optionnel).
3. Appuyez sur **« Enregistrer »**.

### 6.3 La fiche client

Chaque client a une fiche complète avec 3 onglets :

| Onglet | Contenu |
|--------|---------|
| **Info** | Nom, téléphone, email, date d'anniversaire, notes, tier de fidélité |
| **Achats** | Historique complet de toutes les ventes avec dates et montants |
| **Compte** | Solde de dette, historique des paiements et crédits |

### 6.4 Gérer les dettes clients

- **Ajouter une dette** : Fiche client → « Ajouter une dette » → Montant + Motif.
- **Enregistrer un règlement** : Fiche client → « Enregistrer un règlement » → Montant payé.
- Le **solde restant** est calculé automatiquement.

> 💡 **Exemple** : Fatou doit 15 000 FCFA. Elle vient payer 10 000 FCFA. Vous enregistrez le règlement → son solde passe à 5 000 FCFA automatiquement.

### 6.5 Le programme de fidélité

Stockman classe automatiquement vos clients en tiers :

| Tier | Critère | Avantage typique |
|------|---------|-----------------| 
| 🥉 **Bronze** | Nouveau client | Client standard |
| 🥈 **Argent** | Client régulier | Petite remise personnalisée |
| 🥇 **Or** | Client fidèle | Remise spéciale, priorité |
| 💎 **Platine** | Meilleur client | Les meilleurs avantages |

**Pour activer le programme :**
1. **Plus** → **Paramètres** → **Fidélité**.
2. Définissez le **ratio de points** : ex. 1 point pour 1 000 FCFA d'achat.
3. Définissez le **seuil de récompense** : ex. 100 points = 5 000 FCFA de remise.

### 6.6 Promotions

Créez des promotions pour vos clients :

1. Ouvrez **CRM**, puis appuyez sur la carte rapide **Créer une promotion**.
2. Donnez un titre (ex : "Fidélité Or - 10% de remise").
3. Définissez la remise (en % ou en points).
4. Enregistrez la promotion.

### 6.7 Campagnes WhatsApp & SMS

Envoyez des messages promotionnels à vos clients :
1. Ouvrez **CRM**, puis appuyez sur la carte rapide **Campagne WhatsApp**.
2. Choisissez l'audience : tous, un palier spécifique (ex : clients Or), ou des clients individuels.
3. Rédigez votre message avec des **variables** : `{Nom}`, `{points}`.
4. Envoyez.

> 💡 **Exemple de message** : "Bonjour {Nom}, vous avez {points} points de fidélité ! Venez les utiliser avant le 31 mars pour bénéficier de 10% de remise. Merci de votre confiance — Boutique Amina"

### 6.8 Anniversaires Clients

Stockman vous rappelle automatiquement les **anniversaires** de vos clients :
- Un rappel apparaît sur le Dashboard le jour J.
- Envoyez un message personnalisé depuis la fiche client.

## Chapitre 7 — Fournisseurs & Commandes

### 7.1 Ajouter un fournisseur

1. **Plus** → **Fournisseurs**.
2. Appuyez sur **« + »** pour ajouter un fournisseur.
3. Remplissez : Nom, Téléphone, Email, Adresse, Délai de livraison, Conditions de paiement, Notes.

### 7.2 Journal d'interaction fournisseur

Pour chaque fournisseur, gardez un historique de vos échanges :
1. Fiche fournisseur → **« Journal »** → **« + Note »**.
2. Notez l'échange (appel, visite, email).
3. Toutes les interactions sont horodatées.

### 7.3 Lier des produits à un fournisseur

1. Fiche fournisseur → **« Produits liés »** → **« Lier un produit »**.
2. Sélectionnez les produits que fournit ce prestataire.
3. Notez le **prix fournisseur** et la **référence fournisseur**.
4. Cochez **« Fournisseur préféré »** pour qu'il soit priorisé lors des suggestions de réapprovisionnement.

### 7.4 Passer une commande fournisseur

1. **Plus** → **Commandes** → **« + Nouvelle commande »**.
2. Sélectionnez le fournisseur.
3. Ajoutez les produits à commander avec les quantités.
4. Définissez la date de livraison prévue.
5. Validez la commande.

**Scan de facture fournisseur par IA :**
- Prenez en photo la **facture papier de votre fournisseur**.
- L'IA extrait automatiquement les produits et quantités.
- Validez et corrigez si nécessaire.

### 7.5 Suivre les commandes

Chaque commande passe par des étapes :

```
📝 Brouillon → 📤 Envoyée → 🚚 En transit → ✅ Livrée
```

Vous pouvez faire avancer le statut manuellement ou le fournisseur peut le faire depuis son propre compte Stockman.

Sur mobile, les commandes principales (Importer une facture, Exporter, Guide, Nouvelle commande) restent visibles en haut de l'écran.
Le panneau **« Filtres »** sert uniquement à afficher ou masquer les filtres avancés, sans masquer la liste des commandes.

### 7.6 Réception partielle de marchandise

Si le fournisseur livre une partie de la commande :
1. Ouvrez la commande → **« Réception partielle »**.
2. Indiquez les quantités reçues pour chaque produit.
3. Le stock est automatiquement mis à jour.
4. La commande reste ouverte pour le reliquat.

### 7.7 Retours & Avoirs

Si vous devez retourner de la marchandise à un fournisseur, ou si un client vous retourne un article :

**Retour fournisseur :**
1. **Commandes** → onglet **« Retours »** → **« Nouveau retour »**.
2. Sélectionnez la commande d'origine.
3. Choisissez les articles à retourner et la quantité.
4. Indiquez la raison (défaut, surplus, erreur).
5. Un **avoir** (crédit note) est généré automatiquement.

**Retour client :**
1. **Commandes** → onglet **« Retours »** → **« Retour client »**.
2. Sélectionnez le client et les articles retournés.
3. Un avoir est créé dans le compte du client.

> 💡 **Exemple** : Vous recevez 10 caisses de jus mais 2 sont abîmées. Vous faites un retour partiel de 2 caisses. Le fournisseur génère un avoir de 12 000 FCFA que vous déduirez de votre prochaine commande.

### 7.8 Coffre-fort de factures

Conservez toutes vos factures fournisseurs numériquement :
1. Fiche fournisseur → **« Coffre de factures »**.
2. Photographiez ou importez vos factures papier.
3. Elles sont stockées et accessibles à tout moment.

### 7.9 La Marketplace

La Marketplace vous connecte directement avec des fournisseurs enregistrés sur Stockman :
- **Parcourez** les catalogues des grossistes.
- **Filtrez** par catégorie, ville, note, fourchette de prix.
- **Commandez** directement depuis l'application.
- **Évaluez** votre fournisseur après la livraison pour aider la communauté.

## Chapitre 10 — Gérer votre Équipe

### 10.1 Ajouter un employé

Si votre formule autorise plusieurs utilisateurs :

1. **Plus** → **Paramètres** → **Utilisateurs & Permissions** → **« + Inviter »**.
2. Renseignez le nom et l'email de l'employé.
3. **Définissez les permissions** pour chaque module :

| Module | Lecture (📖) | Écriture (✏️) | Aucun (🚫) |
|--------|:-----------:|:------------:|:---------:|
| **Stock** | Voir les produits | Ajouter/Modifier produits | Pas d'accès |
| **Ventes** | Voir les ventes | Créer des ventes (POS) | Pas d'accès |
| **Comptabilité** | Voir les finances | Ajouter des dépenses | Pas d'accès |
| **CRM** | Voir les clients | Gérer les clients | Pas d'accès |
| **Commandes** | Voir les commandes | Créer/Modifier commandes | Pas d'accès |
| **Équipe** | Voir les membres | Gérer les membres | Pas d'accès |

4. Partagez les identifiants par **WhatsApp** avec l'employé.

### 10.2 Modèles de rôles prédéfinis

Pour gagner du temps, Stockman propose des modèles prêts à l'emploi :

- **Vendeur** : POS (écriture) + Stock (lecture).
- **Magasinier** : Stock (écriture) + Commandes (écriture).
- **Comptable** : Comptabilité (écriture) + Ventes (lecture).
- **Superviseur** : Tous les modules en lecture.
- **Manager** : Tous les modules en écriture (sauf Équipe).

### 10.3 Historique d'activité

Suivez ce que fait chaque membre de l'équipe dans **Plus** → **Activité** :

- Chaque action est enregistrée avec la date, l'heure, l'utilisateur et le détail.
- Filtrez par module (stock, caisse, CRM, comptabilité).
- Filtrez par utilisateur.
- Filtrez par période.
- Si vous gérez plusieurs boutiques, vérifiez toujours la boutique active avant d'interpréter une action.

> 💡 **Exemple** : Vous constatez un écart dans votre stock. Vous consultez l'historique d'activité et vous voyez que votre vendeur a créé une sortie de stock de 5 unités à 14h30. Vous pouvez lui demander des explications.

### 10.4 Modifier les permissions ou désactiver un compte

1. **Paramètres** → **Utilisateurs** → Sélectionnez l'employé.
2. Modifiez les permissions ou appuyez sur **« Désactiver »** pour suspendre l'accès.

## ❓ FAQ — Questions Fréquentes

**Q : L'application fonctionne-t-elle sans internet ?**
R : Oui ! Stockman fonctionne en mode hors-ligne. Vos données se synchronisent automatiquement dès que la connexion revient.

**Q : Mes données sont-elles en sécurité ?**
R : Absolument. Vos données sont chiffrées et stockées dans le cloud. Seul vous (et les membres de votre équipe autorisés) pouvez y accéder.

**Q : Puis-je utiliser Stockman en Wolof ?**
R : Oui ! Stockman est disponible en 15 langues dont le Wolof, le Peul, l'Arabe et le Français. Changez la langue dans les Paramètres.

**Q : Comment créer une facture professionnelle ?**
R : Allez dans Comptabilité → « Facture » → « Créer une facture ». Remplissez les lignes, ajoutez la TVA si besoin, et générez le PDF à partager par email ou WhatsApp.

**Q : Qu'est-ce que le Grand Livre ?**
R : C'est le registre complet de toutes vos opérations (ventes, entrées stock, dépenses, ajustements). Accédez-y depuis le Dashboard → bouton « Historique ». Vous pouvez l'exporter en CSV pour votre comptable.

**Q : Comment gérer les produits en lot avec date de péremption ?**
R : Lors de l'ajout ou la modification d'un produit, renseignez le N° de lot et la date de péremption. Stockman vous alertera automatiquement 7 jours avant l'expiration.

**Q : Comment exporter mes données pour mon comptable ?**
R : Depuis la Comptabilité, appuyez sur « Export CSV » pour exporter les ventes et dépenses. Depuis le Dashboard → Historique, exportez le Grand Livre complet.

**Q : Comment contacter le support ?**
R : Dans l'application : **Plus** → **Paramètres** → **Aide & Support**. Consultez d'abord le guide du module concerné, puis ouvrez un ticket si nécessaire. Le canal de réponse dépend de votre formule.

**Q : Que se passe-t-il après les 90 jours d'essai ?**
R : Vous pouvez souscrire à un plan payant. Si vous ne renouvelez pas, vos données restent sauvegardées mais certaines fonctionnalités avancées sont désactivées.

**Q : Un de mes employés a fait une erreur dans le stock. Comment la corriger ?**
R : Allez dans **📦 Produits** → Sélectionnez le produit → **« -Stock »** ou **« +Stock »** → Entrez la quantité à corriger et la raison **« Ajustement inventaire »**. Tout sera tracé dans le Grand Livre.
