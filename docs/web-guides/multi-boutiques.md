# Guide - Vue multi-boutiques

## 1. Role du module

La vue multi-boutiques permet de comparer plusieurs boutiques d'une meme organisation dans un cadre consolide.

Profils concernes : utilisateurs autorises sur l'organisation, selon le plan et les droits.

## 2. Acces

Barre laterale -> **Multi-boutiques** quand l'acces est autorise.

## 3. Ce que l'ecran montre

Selon les droits et les donnees disponibles, l'ecran peut afficher :

- indicateurs consolides ;
- comparaison par boutique ;
- chiffres d'affaires ;
- ventes ;
- stock ;
- ruptures ;
- rotation ;
- bascule rapide vers une boutique active.

## 4. Regles de visibilite

Cet ecran doit suivre le plan :

- visible si le compte a droit a la vue multi-boutiques ;
- masque sinon.

Il ne doit pas etre montre a un compte qui n'a pas acces a cette fonction.

## 5. Questions frequentes

| Question | Reponse |
|---|---|
| Pourquoi je ne vois pas la vue multi-boutiques ? | Elle depend du plan, du nombre de boutiques et des droits de l'utilisateur. |
| Peut-on changer de boutique depuis cet ecran ? | Oui, si l'action est disponible dans l'organisation. |
| Les indicateurs sont-ils consolides ? | Oui, l'objectif est justement de comparer et de piloter plusieurs boutiques. |
