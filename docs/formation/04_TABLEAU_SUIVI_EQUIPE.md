# Tableau de suivi equipe - Modele

Ce document propose un modele simple de tableau de suivi pour piloter 5 stagiaires commerciaux.

Le tableau peut etre tenu :
- dans Google Sheets ;
- dans Excel ;
- dans Notion ;
- ou plus tard dans un outil interne.

---

## 1. Vue quotidienne recommandee

| Date | Stagiaire | Zone | Visites | Discussions utiles | Demonstrations | Suivis WhatsApp obtenus | Installations lancees | Relances faites | Objection principale | Prochaine action cle | Commentaire manager |
|------|-----------|------|---------|--------------------|----------------|--------------------------|-----------------------|-----------------|----------------------|----------------------|---------------------|
|      |           |      |         |                    |                |                          |                       |                 |                      |                      |                     |

---

## 2. Vue hebdomadaire recommandee

| Semaine | Stagiaire | Type de mission | Visites totales | Discussions utiles | Demonstrations | Suivis obtenus | Installations lancees | Relances prevues | Relances faites | Points forts | Point a corriger |
|---------|-----------|-----------------|-----------------|--------------------|----------------|----------------|-----------------------|------------------|-----------------|-------------|------------------|
|         |           |                 |                 |                    |                |                |                       |                  |                 |             |                  |

---

## 3. Vue equipe manager

| Semaine | Total visites | Total demonstrations | Total suivis | Total installations lancees | Top besoin identifie | Top objection | Zone la plus active | Zone la moins active | Action manager de la semaine |
|---------|---------------|----------------------|--------------|-----------------------------|----------------------|---------------|---------------------|----------------------|------------------------------|
|         |               |                      |              |                             |                      |               |                     |                      |                              |

---

## 4. Colonnes importantes a ne pas supprimer

- stagiaire
- zone
- visites
- discussions utiles
- demonstrations
- suivis obtenus
- prochaine action
- objection principale

Ces colonnes suffisent deja a piloter correctement si l'equipe debute.

---

## 5. Regles de mise a jour

- Le stagiaire remplit sa partie tous les jours.
- Le responsable relit et corrige en fin de journee ou le lendemain matin.
- La vue hebdomadaire est consolidee une fois par semaine.
- Les commentaires manager doivent etre concrets et courts.

---

## 6. Ce que le tableau doit permettre de voir vite

- qui travaille vraiment sur le terrain ;
- qui transforme ses visites en echanges utiles ;
- qui obtient des prochaines etapes concretes ;
- qui relance bien ;
- quelles objections reviennent le plus ;
- quelles zones meritent plus d'effort ou un autre discours.
