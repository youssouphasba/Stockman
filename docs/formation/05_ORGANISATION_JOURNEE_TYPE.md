# Organisation d'une journee type - Stagiaire terrain

Ce document sert de repere simple pour aider un stagiaire a structurer sa journee sur le terrain.

Le but est d'eviter :
- les journees desorganisees ;
- les visites faites sans preparation ;
- les relances oubliees ;
- les comptes rendus incomplets.

---

## 1. Avant de partir

Chaque stagiaire doit verifier :
- telephone charge
- application prete
- compte de demonstration accessible
- connexion et contacts disponibles si necessaire
- fiche de prospection ou formulaire pret
- objectif du jour connu
- zone du jour confirmee

---

## 2. Matin - Preparation et premier passage terrain

### 08h30 - 09h00

- revoir les objectifs du jour
- verifier les relances prioritaires
- choisir l'ordre de passage dans la zone

### 09h00 - 12h30

- faire les premieres visites
- privilegier les commerces accessibles et les heures pertinentes
- noter chaque echange utile juste apres la visite

Regle :
- ne pas enchainer 10 visites sans rien noter

---

## 3. Pause et point a mi-journee

### 12h30 - 13h30

Le stagiaire fait un mini-bilan :
- nombre de visites
- nombre de discussions utiles
- nombre de demonstrations
- blocages rencontres
- relances urgentes a faire

Si besoin, il ajuste sa zone ou son discours pour l'apres-midi.

---

## 4. Apres-midi - Deuxieme bloc terrain

### 13h30 - 17h00

- reprendre les visites
- faire les rappels promis
- relancer les prospects les plus chauds
- confirmer les prochaines etapes

Le stagiaire doit garder un objectif simple :
- faire avancer les prospects utiles, pas seulement augmenter le nombre de visites.

---

## 5. Fin de journee

### 17h00 - 18h00

- terminer les fiches terrain
- envoyer les suivis WhatsApp promis
- signaler les cas importants
- mettre a jour le tableau de suivi
- preparer les relances du lendemain

---

## 6. Regles de discipline utiles

- ne jamais promettre un rappel puis oublier ;
- ne pas laisser s'accumuler les fiches non remplies ;
- ne pas passer toute la journee sur un seul prospect ;
- ne pas confondre activite et performance ;
- toujours finir la journee avec un suivi propre.

---

## 7. Regle simple

Une bonne journee terrain se juge sur 4 points :
- preparation ;
- qualite des echanges ;
- clarte des prochaines etapes ;
- qualite du reporting.
